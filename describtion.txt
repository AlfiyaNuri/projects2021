Дашборды в Google Data Studio

Задание 2
Знакомство с основными методами анализа данных с помощью библиотеки Pandas, анализируя набор данных по оттоку клиентов телеком-оператора.

Текст задания
Решение

Построение дашбордов в Google Data Studio, чтобы увидеть общую ситуацию с продажами/ценами/прибылью компании, продающей продукты по всему миру.
Анализ оттока клиентов телеком-оператора с помощью стандартных инструментов бибилиотеки Pandas

Прогнозирование оттока клиентов телеком-оператора, используя методы машинного обучения. 
Был проведен исследовательский анализ данных с помощью бибилиотек DataPrep и Pandas profiling
Использовались библиотеки Scikit-learn, XGBoost, AutoGluon, PyCaret.  
Затем было сделано сравнение метрик разных прогнозов, чтобы понять какой метод точнее прогнозирует.

Анализ причин увольнения сотрудников. 
Первоначально анализ был проведен с помощью SQL запросов в блокноте Count.co
Затем в BigQuery было произведено прогнозирование увольнения с помощью машинного обучения методами логистической регрессии и XQBoost.
Затем в Python было произведено прогнозирование увольнения с использованием бибилиотеки PyCaret.
В заключении было сделано сравнение метрик разных вариантов машинного обучения.
